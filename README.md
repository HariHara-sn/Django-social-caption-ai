

# **Django-LLM-PostGen**



**Django-LLM-PostGen** is a backend API that generates short, platform-specific social media posts using Google’s Gemini LLM (`gemini-2.5-flash`). It’s built with Django scalability, maintainability.



This project is a **social media content generator API** by a LLM. Its main purpose is to create concise, tailored captions for platforms like Instagram, LinkedIn, Twitter, and Facebook based on a user-provided context term (e.g., “Business”, “Growth”, “Motivation”).

### Key Features

* **Platform-specific posts:** Generates captions optimized for Instagram, LinkedIn, Twitter, and Facebook.
* **Controlled output:** Ensures responses are concise (e.g., 2 sentences), emoji-free, and formatted correctly.

* **LLM integration:** Uses Google Gemini API (`gemini-2.5-flash`) for content generation.

### Tech Stack

* **Backend:** Django, Django REST Framework
* **LLM:** Google Gemini API (`gemini-2.5-flash`)
* **Testing:** Postman (API testing)
